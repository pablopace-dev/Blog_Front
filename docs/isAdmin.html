(async, static) isAdmin(req, res, next)

Primero se comprueba que el token almacenado sea válido y luego, comprueba si el usuario tiene rol de 'admin', si es así, continúa al siguiente paso, si no, se redigire al 'index' del usuario. En cualquier de los 2 casos, el token se renueva. Si el token no es válido, se redirige a 'login'.
