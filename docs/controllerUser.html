(async, static) loginUser(req, res)

Recibe los datos para validar primero, y luego loguear al usuario. Si las validaciones son correctas se redirige al 'index' de usuario o administrador según el rol del usuario, y se crean 2 cookies, una con el token y otros con los datos del usuario (id, name, email, rol). En caso de errores se devuelven en un Object, y se renderiza el 'login'.
